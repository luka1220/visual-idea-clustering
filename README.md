## Visual idea Clustering

### Run:

- `npm install`
- `npm init`
- `npm start`

### Tests:

- `npm test`

### Related Work

- [IdeaHound](http://www.eecs.harvard.edu/~kgajos/papers/2016/siangliulue16ideahound-uist.shtml)
